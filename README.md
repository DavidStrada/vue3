# New in Vue3

> Minimal webpack setup for Vue 3 (alpha)

### Topics
---

- Global Mounting
- Fragments
- Suspense
- Portals
- Composition API

### Prerequisites
- Node & NPM

### Install
```sh
npm install
```
### Usage
##### Develop
```sh
# run dev server at localhost:8080
npm run dev
```
##### Build
```sh
# transpile js for deployment
npm run build
```
